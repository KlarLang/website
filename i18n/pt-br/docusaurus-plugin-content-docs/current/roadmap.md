Klar está atualmente em fase pré-1.0.

## Versionamento

* v0.x: instável, mudanças que quebram a compatibilidade (breaking changes) são permitidas
* v1.0.0: primeira linha estável (objetivo)

## Prioridades (planejado)

1. Interoperabilidade JSON (primeiro marco real de interoperabilidade)
2. Validação semântica mais profunda (validadores de intenção/rigor)
3. Relatório de múltiplos diagnósticos
4. Backends/alvos adicionais (posteriormente)

Este roadmap é uma direção, não uma garantia.

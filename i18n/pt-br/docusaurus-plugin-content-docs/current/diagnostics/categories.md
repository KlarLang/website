# Categorias

## LEXICAL
Falha na tokenização (caracteres inválidos, literais malformados).

## SYNTAX
Falha no parser (construtos ausentes, estrutura incorreta).

## SEMANTIC
Falha nas verificações de tipo/símbolos (símbolo indefinido, retorno ausente, redeclaração).

## BACKEND
Falha na geração do alvo/saída (alvo desconhecido, recurso não suportado pelo backend).

## CLI
Falha na entrada/uso pelo usuário (tipo de arquivo inválido, arquivo ausente, etc.).

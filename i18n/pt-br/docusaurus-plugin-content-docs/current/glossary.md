# Glossário

## Transpilador

Um compilador que emite código-fonte de outra linguagem (Klar → código-fonte Java).

## Backend

O alvo de geração de código (atualmente: Java).

## Diagnóstico

Um relatório de erro estruturado: código + span (intervalo) + causa + correção.

## Estrito por padrão

A linguagem rejeita comportamentos ambíguos ou implícitos, a menos que sejam explicitamente escritos.

# Limitações da Linguagem (atualmente)

Estas são funcionalidades atualmente ausentes:

- arrays (planejado)
- structs / tipos definidos pelo usuário
- sistema de módulos (import é placeholder atualmente)
- vinculação/importação de símbolos entre arquivos
- conversões de tipo (casts)
- conversões numéricas implícitas
- switch/for/foreach

Esta lista não é "vergonha". É um contrato: Klar documenta o que existe.

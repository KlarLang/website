# Funções Integradas

Funções integradas existem sem importações.

Suportadas atualmente:
- `print(...)`
- `println(...)`
- `printf(...)` (atualmente placeholder)

---

## Observações

`printf` existe na linguagem de superfície atualmente, mas pode não estar totalmente implementada ainda.
Se você quer saída confiável: prefira `println`.

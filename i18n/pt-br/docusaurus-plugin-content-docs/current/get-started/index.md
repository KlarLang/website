# Primeiros passos

Este é o caminho mais curto para executar o Klar.

1) Instale o Klar via Loom  
2) Execute o `Hello World`  
3) Aprenda o essencial: tipos, funções, retornos e controle de fluxo

O Klar é rigoroso. O objetivo é confiabilidade, não conveniência.

# Linguagem

Esta seção descreve a linguagem Klar **como implementada atualmente**.

Klar é projetada para ser:
- explícita
- determinística
- rigorosa por padrão

Se você está procurando por "recursos de conveniência", Klar parecerá hostil. Isso é intencional.

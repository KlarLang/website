# Autor

Klar é um projeto independente de design de linguagem criado e mantido por [~K'](https://github.com/lucaas-d3v).

O projeto é movido pelo interesse em:

- design de linguagens e semântica
- intenção explícita em sistemas de software
- arquiteturas poliglotas e de longa duração
- pipelines de compiladores e diagnósticos

A Klar não é apoiada por uma empresa.
Não foi projetada para ser orientada pelo mercado.
Suas decisões de design refletem escolhas deliberadas (trade-offs),
não tentativas de adoção em massa.

Este documento existe para tornar a autoria explícita.
